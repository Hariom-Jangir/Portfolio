## 🧠 Project Overview

This is a **full-stack personal portfolio website** built to showcase my skills, projects, and provide a way for recruiters or visitors to contact me directly.

The frontend features a modern UI with **smooth animations and an interactive 3D background using Three.js**, while the backend handles contact form submissions through a REST API.

---

## 🏗️ Architecture

Frontend (React + Vite) → API (/contact) → Backend (Node.js + Express)

## 🛠️ Tech Stack

### 🎨 Frontend

* React.js (Vite)
* Tailwind CSS
* Framer Motion (animations)
* Three.js (3D graphics & interactive background)

### ⚙️ Backend

* Node.js
* Express.js
* REST API (`POST /contact`)

### ☁️ Deployment

* Frontend → Vercel
* Backend → Render

## 📬 Contact API

### Endpoint:

```bash
POST /contact
```

### Request Body:

```json
{
  "name": "Your Name",
  "email": "your@email.com",
  "message": "Your message"
}
```

### Response:

```json
{
  "success": true
}
```

## 📦 Installation & Setup

### Clone the repository

```bash
git clone https://github.com/Hariom-Jangir/Portfolio.git
cd Portfolio
```

### Install frontend dependencies

```bash
npm install
npm run dev
```

### Setup backend

```bash
cd backend
npm install
node server.js
```
